# Phase 8.5 — 한국어 문제 인식 강화

Phase 8.5는 서비스화 전에 한국어 문제 입력 품질을 높이기 위한 전용 단계입니다.
기존 Phase 8까지는 한국어 키워드 기반 분류가 가능했지만, 한국어 교재식 표현과 단위 변환에서 오탐/누락이 있었습니다.

## 추가된 핵심 기능

- 한국어 단위 변환 강화
  - `500g` → `0.5 kg`
  - `30cm` → `0.3 m`
  - `5,000cm` → `50 m`
  - `72 km/h` → `20 m/s`
  - `2분` → `120 s`
  - `m/s²` → `m/s^2`
  - `kgm^2` → `kg*m^2`
- 한국어 암시 조건 처리
  - `정지 상태에서 출발`, `가만히 있다가` → `v0 = 0`
  - `멈출 때까지`, `최종적으로 정지` → `vf = 0`
- 구할 값 추출 개선
  - `걸리는 시간은?` → time target
  - `최종속도를 구하라` → final velocity target
  - `이동거리/변위/사거리` → distance target
- 한국어 마찰 없음 표현 강화
  - `마찰이 없는`, `마찰 없음`, `마찰을 무시`, `매끄러운`
- 한국어 자연문장 테스트 추가
  - `test_phase8_5_korean_parser.py`

## 검증된 한국어 입력 예시

```text
정지 상태에서 출발한 물체가 가속도 2m/s²로 5초 동안 직선 운동한다. 최종속도를 구하라.
→ constant_acceleration_1d, v_f = 10 m/s
```

```text
발사속도 72 km/h, 발사각 30도인 포물선 운동에서 같은 높이에 착지할 때 사거리를 구하라.
→ projectile_motion, R ≈ 35.312 m
```

```text
스프링 상수 200N/m, 질량 500g인 스프링-질량계의 주기를 구하라.
→ spring_mass_vibration, T ≈ 0.314 s
```

```text
도르래 관성모멘트 I=0.12 kgm^2, 도르래 반지름 30cm, m1=2kg, m2=5kg인 질량 있는 도르래에서 가속도를 구하라.
→ massive_pulley_atwood
```

## 남은 한계

- 그림 문제 자동 해석은 아직 지원하지 않습니다.
- 조사/어순이 매우 자유로운 장문 교재 문장은 아직 완전하지 않습니다.
- 방향이 포함된 벡터 문제는 기본형 위주입니다.
- `오른쪽으로`, `왼쪽으로`, `위로`, `아래로` 같은 방향 부호 처리는 아직 제한적입니다.

다음 서비스화 단계 전에 한국어 문제 corpus를 계속 추가하면서 파서를 확장하면 됩니다.
