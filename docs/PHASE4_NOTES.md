# DynaTutor Phase 4 Notes

Phase 4 adds advanced dynamics tools that are common in sophomore dynamics:

- Polar kinematics: `v = r_dot e_r + r theta_dot e_theta`, `a_r = r_ddot - r theta_dot^2`, `a_theta = r theta_ddot + 2 r_dot theta_dot`
- Instant center velocity method: `v = omega r`
- Slot-pin relative motion: relative sliding plus rotating slot components
- Basic plane rigid body velocity: `v_B = v_A + omega x r_B/A`

The old uploaded app is still treated only as legacy hints. The main architecture remains canonical representation -> solver registry -> verification -> UI cards.
