# Claude Code 또는 두 번째 모델 리뷰 템플릿

```text
이 PR은 DynaTutor 동역학 엔진의 goal 문서를 구현한 변경이야.

코드를 바로 수정하지 말고 우선 독립 리뷰만 해줘.

다음을 확인해:
1. goal과 acceptance criteria를 실제로 충족하는가
2. 변경 범위가 goal 밖으로 퍼졌는가
3. 변수 이름, 배열 순서, 첫 결과에 의존한 선택이 남아 있는가
4. 자연어 근거 없이 값이나 가정을 생성하는가
5. 단위, 좌표계, 부호가 일관적인가
6. 복수 해, 모호성, 특이점, 수치 불안정성을 안전하게 처리하는가
7. 적용되지 않는 보존법칙을 잘못 강제하는가
8. 테스트가 구현과 같은 오류를 공유하는 파생형 테스트뿐인가
9. 독립 oracle 또는 metamorphic test가 충분한가
10. 기존 API와 학생용 풀이를 깨뜨릴 위험이 있는가
11. 외부 선택 의존성이 일반 런타임을 깨뜨리는가
12. 성능 회귀 또는 flaky 가능성이 있는가

각 문제를 다음 형식으로 알려줘:
- 심각도
- 파일과 위치
- 왜 문제인지
- 재현 방법
- 권장 수정
- 필요한 회귀 테스트

마지막에 merge 가능 / 수정 후 merge / 재설계 필요 중 하나로 판단해줘.
```
